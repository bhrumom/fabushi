## 🎯 使用方法

### 快速开始
1. 访问主页面，系统会自动请求通知权限

4. 锁屏或切换应用，在通知栏查看进度

## 🆘 故障排除

### 常见问题解决
1. **通知不显示** → 检查浏览器通知权限
2. **进度不更新** → 刷新页面重新初始化
3. **点击无反应** → 检查是否有多个标签页
4. **权限被拒绝** → 在浏览器设置中手动开启

### 调试工具
- 使用 `/notification-test.html` 页面测试
- 查看浏览器开发者工具的Console日志
- 检查Application → Service Workers状态
