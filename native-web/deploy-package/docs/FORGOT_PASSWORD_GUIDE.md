# 忘记密码功能使用指南

## 功能概述

全球法布施系统现已支持完整的忘记密码功能，用户可以通过邮箱重置密码。

## 功能特点

- ✅ 安全的邮箱验证流程
- ✅ 30分钟有效期的重置链接
- ✅ 强密码要求（8位+大小写+数字）
- ✅ 用户友好的界面设计
- ✅ 完整的错误处理

## 使用流程

### 1. 访问忘记密码页面
- 在登录页面点击"忘记密码？"链接
- 或直接访问 `/forgot-password.html`

### 2. 输入邮箱地址
- 输入注册时使用的邮箱地址
- 点击"发送重置链接"按钮

### 3. 检查邮箱
- 查收重置密码邮件（包括垃圾邮件文件夹）
- 邮件包含重置链接，30分钟内有效

### 4. 重置密码
- 点击邮件中的重置链接
- 输入符合要求的新密码
- 确认新密码
- 点击"重置密码"按钮

### 5. 登录验证
- 使用新密码登录系统

## 密码要求

新密码必须满足以下条件：
- 至少8个字符
- 包含大写字母（A-Z）
- 包含小写字母（a-z）
- 包含数字（0-9）

## 安全特性

1. **时效性**：重置链接30分钟后自动失效
2. **一次性**：每个重置链接只能使用一次
3. **邮箱验证**：只有注册邮箱才能接收重置链接
4. **强密码**：新密码必须符合安全要求

## 常见问题

### Q: 没有收到重置邮件怎么办？
A: 
- 检查垃圾邮件文件夹
- 确认输入的邮箱地址正确
- 等待几分钟后重试
- 点击"重新发送"按钮

### Q: 重置链接显示无效或已过期？
A: 
- 重置链接有效期为30分钟
- 每个链接只能使用一次
- 请重新申请重置链接

### Q: 新密码不符合要求？
A: 确保密码包含：
- 至少8个字符
- 大写字母
- 小写字母  
- 数字

## 技术实现

### 前端页面
- `login.html` - 登录页面（包含忘记密码链接）
- `forgot-password.html` - 忘记密码申请页面
- `reset-password.html` - 重置密码页面

### API端点
- `POST /api/auth/forgot-password` - 发送重置邮件
- `POST /api/auth/reset-password` - 重置密码
