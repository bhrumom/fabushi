## 1. 软件概述

### 1.1 软件简介

大乘软件是一款基于Flutter框架开发的跨平台移动应用软件，旨在通过现代技术手段实现佛教经文内容的全球传播与共享。本软件采用云端架构设计，集成了Cloudflare Workers边缘计算后端服务，为用户提供安全、高效的内容分发体验。

### 1.2 主要功能

本软件包含以下核心功能模块：

- **用户认证系统**：提供用户注册、登录、密码找回等完整的身份认证服务
- **会员管理系统**：支持多层级会员体系，包含兑换码系统和支付集成
- **全球传输功能**：实现经文内容的全球化传输，支持实时进度显示
- **在线人数统计**：基于WebSocket技术的实时在线用户统计
- **社交互动功能**：内容点赞、收藏与分享功能
- **禅修室功能**：计时器、经文选择、修习记录统计

### 1.3 软件特点

- **跨平台支持**：一套代码同时支持Android、iOS、Web、macOS、Windows、Linux六大平台
- **实时通信**：采用WebSocket技术实现毫秒级消息推送
- **安全可靠**：JWT Token认证机制，HTTPS全程加密传输
- **离线可用**：支持内容本地缓存，无网络时可访问已下载内容
- **高性能**：经过深度优化，达到60fps流畅体验

## 2. 运行环境

### 2.1 硬件要求

| 平台 | 最低配置 | 推荐配置 |
|------|----------|----------|
| Android | 2GB RAM, ARM64处理器 | 4GB RAM, 骁龙660及以上 |
| iOS | iPhone 6s及以上 | iPhone 8及以上 |
| Web | 现代浏览器 | Chrome 88+, Safari 14+ |
| macOS | macOS 10.14+ | macOS 12.0+ |
| Windows | Windows 10 64位 | Windows 11 |
| Linux | Ubuntu 18.04+ | Ubuntu 22.04+ |

### 2.2 软件要求

- **Android**：Android 5.0 (API 21) 及以上版本
- **iOS**：iOS 12.0 及以上版本
- **Web**：支持ES6的现代浏览器
- **macOS**：macOS 10.14 Mojave 及以上版本
- **Windows**：Windows 10 版本1903及以上
- **Linux**：支持GNU C Library 2.17及以上的发行版

### 2.3 网络要求

- 需要稳定的互联网连接
- 推荐带宽：2Mbps及以上
- 支持HTTP/HTTPS协议
- 支持WebSocket协议（可选，用于实时功能）

## 3. 安装说明

### 3.1 Android平台安装

1. 从官方渠道下载APK安装包
2. 打开下载的安装包
3. 如提示"未知来源"，请在设置中允许安装
4. 按照提示完成安装
5. 在应用列表中找到"大乘软件"图标，点击启动

### 3.2 iOS平台安装

1. 打开App Store应用商店
2. 搜索"大乘软件"
3. 点击"获取"按钮
4. 使用Face ID/Touch ID或输入Apple ID密码确认
5. 等待下载安装完成后启动应用

### 3.3 Web平台使用

1. 打开现代浏览器（推荐Chrome或Safari）
2. 访问官方网址：https://fabushi.ombhrum.com
3. 等待页面加载完成
4. 可选：添加至主屏幕以获得类似原生应用的体验

### 3.4 桌面平台安装

**macOS：**
1. 下载.dmg安装包
2. 双击打开安装包
3. 将应用拖拽至"应用程序"文件夹
4. 在启动台中找到应用并启动

**Windows：**
1. 下载.exe安装程序
2. 双击运行安装程序
3. 按照安装向导完成安装
4. 在开始菜单或桌面找到快捷方式启动

## 5. 操作指南

### 5.1 首次使用流程

1. **启动应用**
   - 打开应用后进入欢迎页面
   - 等待初始化完成

2. **注册账号**
   - 点击"注册"按钮
   - 填写用户名、邮箱、密码
   - 点击"获取验证码"
   - 输入收到的邮箱验证码
   - 点击"注册"完成

3. **登录账号**
   - 输入用户名或邮箱
   - 输入密码
   - 点击"登录"

4. **开始使用**
   - 登录成功后自动进入首页
   - 浏览3D地球动画
   - 探索各功能模块

### 5.2 全球发送操作

1. **选择经文**
   - 在首页点击"选择经文"按钮
   - 从下拉列表选择要发送的经文
   - 可选择心经、大悲咒、楞严咒等

2. **开始发送**
   - 点击"开始发送"按钮
   - 观察3D地球上的光束动画
   - 顶部显示发送进度条

3. **停止发送**
   - 点击"停止发送"按钮
   - 系统记录本次发送数据

### 5.3 禅修室使用

1. **进入禅修室**
   - 点击底部导航栏"禅修室"
   - 进入禅修主界面

2. **选择经文**
   - 点击经文选择下拉菜单
   - 选择要修习的经文
   - 支持搜索查找

3. **开始禅修**
   - 点击"开始禅修"按钮
   - 计时器自动开始
   - 屏幕显示当前禅修时长

4. **计数操作**
   - 按手机音量+键进行计数
   - 每次按键计数增加1
   - 实时显示当前计数

5. **结束禅修**
   - 点击"结束禅修"按钮
   - 系统自动保存本次记录
   - 显示本次禅修统计

### 5.4 法流浏览操作

1. **进入法流**
   - 点击底部导航栏"法流"
   - 进入内容浏览界面

2. **浏览内容**
   - 上下滑动切换内容
   - 视频内容自动播放
   - 文本内容可开启朗读

3. **点赞收藏**
   - 点击右侧爱心图标点赞
   - 已点赞内容显示红色实心爱心
   - 收藏的内容可在个人中心查看

4. **内容互动**
   - 点击评论图标查看/发表评论
   - 点击分享图标分享内容

### 5.5 个人中心操作

1. **查看信息**
   - 点击底部导航栏"我的"
   - 查看个人信息和统计数据

2. **管理收藏**
   - 点击"我的喜欢"卡片
   - 查看已收藏内容列表
   - 可取消收藏或查看详情

3. **会员管理**
   - 点击会员状态区域
   - 查看当前会员到期时间
   - 可购买/续费会员

4. **设置调整**
   - 点击设置图标
   - 调整各项设置

## 6. 常见问题解答

### 6.1 账号相关

**Q：忘记密码怎么办？**
A：点击登录页面的"忘记密码"链接，输入注册邮箱，通过邮箱验证码重置密码。

**Q：收不到验证码邮件？**
A：请检查垃圾邮件文件夹，或等待1分钟后重新发送。如持续无法收到，请联系客服。

**Q：如何修改密码？**
A：进入"我的"->"设置"->"账户安全"->"修改密码"。

### 6.2 功能相关

**Q：全球发送的流量有什么用？**
A：流量代表您法布施的功德值，会记录在全球排行榜中。

**Q：禅修记录会丢失吗？**
A：不会，禅修记录会本地持久化存储，并可同步到云端。

**Q：点赞的内容在哪里查看？**
A：进入"我的"->"我的喜欢"即可查看所有点赞内容。

### 6.3 技术相关

**Q：应用闪退怎么办？**
A：
1. 检查系统版本是否满足最低要求
2. 清除应用缓存后重试
3. 卸载重装应用
4. 如问题持续，请联系技术支持

**Q：视频无法播放？**
A：
1. 检查网络连接
2. 等待视频加载完成
3. 尝试切换网络（WiFi/移动数据）

**Q：显示"网络连接失败"？**
A：
1. 检查设备网络连接
2. 尝试切换网络
3. 检查是否开启了VPN
4. 稍后重试
