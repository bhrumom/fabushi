# Tests

v1 使用人工文件检查作为验收方式。

## 验收清单

- `agent/README.md` 说明入口流程和目录职责。
- `agent/config/registry.md` 能查到所有员工和协作规则。
- 每个 `agent/agents/{agent_id}/` 都有 `AGENT.md`、`status.md`、`memory.md`、`workspace/README.md`。
- `agent/chatroom/` 有格式说明和当天示例。
- `agent/tasks/` 有任务格式和示例任务。
- `agent/reports/` 有汇报格式和当天示例。
- `agent/handbook/` 有运行手册和 GitHub MCP 规范。
- 仓库中没有新增密钥、token、Cookie 或账号密码。
