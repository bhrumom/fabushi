# Release Notes - 大乘 (Mahayana) v1.1.0

## 🚀 版本特性 (Features)
- **全新名称**: 应用正式更名为“大乘”(Mahayana)，致力于佛教经文的全球传播。
- **跨平台支持**: 支持 Windows (x64) 及 Web 端。
- **UI/UX 优化**: 引入深色太空主题，优化视觉体验。
- **核心功能**:
  - 全球法布施内容流。
  - 智能语音识别与录音。
  - 支付宝支付集成。
  - 离线语音与文本处理。

## 🛠 安装说明 (Installation)
1. 下载下方的 `setup.exe` 或 `Mahayana.msix`。
2. 运行安装程序并按照提示完成安装。
3. 如果系统提示来自未知发布者，请选择“仍要运行”。
