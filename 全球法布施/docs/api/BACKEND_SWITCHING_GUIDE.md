# 后端切换使用指南

## 🎯 功能说明

现在应用支持在**测试模式**和**真实Cloudflare后端**之间动态切换，无需重启应用。

## 🔧 如何切换模式

### 方法1：通过应用内设置
1. **登录状态下**：点击右上角用户头像 → 选择"应用设置"
2. **未登录状态下**：点击右上角设置图标 ⚙️

### 方法2：设置界面操作
1. 进入设置界面
2. 找到"运行模式"卡片
3. 切换"测试模式"开关：
   - **开启**：使用本地模拟数据（无需网络）
   - **关闭**：连接真实Cloudflare后端

## 📱 两种模式对比

### 🧪 测试模式（默认开启）
- ✅ **优点**：无需网络连接，响应快速
- ✅ **适用**：功能测试、界面演示
- ⚠️ **限制**：只能使用预设测试账户
- 📋 **测试账户**：
  - `test@example.com` / `password123` (高级会员)
  - `admin@fabushi.com` / `admin123` (管理员)
  - `user@test.com` / `test123` (普通用户)

### 🌐 真实后端模式（默认关闭）
- ✅ **优点**：可使用真实注册的账户
- ✅ **功能**：完整的用户数据和会员功能
- ⚠️ **要求**：需要稳定的网络连接
- 🔗 **后端**：连接到 `https://fabushi.ombhrum.com`

## 🚀 使用步骤

### 使用真实账户登录
1. 确保应用处于**真实后端模式**（测试模式开关关闭）
2. 点击"开始使用"进入登录界面
3. 输入您之前注册的真实账户信息
4. 登录成功后可以访问完整功能

### 使用测试账户体验
1. 切换到**测试模式**（测试模式开关开启）
2. 使用上述测试账户登录
3. 体验所有功能（数据为模拟数据）

## 🔄 切换注意事项

1. **切换后立即生效**：无需重启应用
2. **登录状态保持**：切换模式不会自动登出
3. **数据隔离**：两种模式的数据完全独立
4. **网络要求**：真实模式需要网络，测试模式不需要

## 🐛 故障排除

### 网络连接错误
- **现象**：`HandshakeException: Connection terminated during handshake`
- **解决**：切换到测试模式，或检查网络连接

### 登录失败
- **测试模式**：确保使用正确的测试账户
- **真实模式**：确保账户已在后端注册

### 设置不生效
- 尝试热重载应用（在终端按 `r` 键）
- 或重启应用

## 📊 当前状态查看

在设置界面的"当前状态"卡片中可以查看：
- 运行模式（测试/生产）
- 后端地址
- 数据来源
