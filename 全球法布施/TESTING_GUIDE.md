## 📱 热重载步骤

1. **找到运行Flutter的终端窗口**
   - 应该显示类似：`Flutter run key commands.`
   - 以及 `r Hot reload. 🔥🔥🔥`

2. **执行热重载**
   - 在终端中直接按 `r` 键
   - 按回车键
   - 等待看到 `Performing hot reload...` 消息

3. **验证热重载成功**
   - 应该看到 `Reloaded X of Y libraries in XXXms.`
   - 应用界面会自动更新

## 🐛 常见问题

### 网络连接错误
- **问题**: `HandshakeException: Connection terminated during handshake`
- **解决**: 热重载后应该消失（已启用测试模式）

### 编译错误
- **解决**: 重新启动应用 `flutter run -d macos -t lib/main_simple_test.dart`

### 界面显示问题
- **解决**: 尝试热重启 `R` 键而不是热重载 `r` 键
